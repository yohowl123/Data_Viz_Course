# 📊 數據視覺化課程：從基礎到進階 (Data Viz Learning)

這是一個記錄我數據視覺化學習歷程的專案。包含使用 **Python (Seaborn/Plotly)** 與 **R (ggplot2)** 的實作練習。

---

## 📂 資料夾配置說明

- **`Weekly_Assignments/`**：每週的實作作業。
  - `Week_XX/Python/`：存放 `.ipynb` 或 `.py` 繪圖腳本。
  - `Week_XX/R/`：存放 `.R` 或 `.Rmd` 繪圖腳本。
- **`data/`**：存放課程練習用的原始數據（CSV, Excel 等）。
- **`outputs/`**：存放產出的視覺化成品（圖片或互動圖表）。
- **`docs/`**：課程筆記與 NotebookLM 輔助生成的技術摘要。

---

## 🛠️ 開發工具與環境

- **語言**: Python 3.x, R 4.x
- **主要套件**:
  - Python: `pandas`, `matplotlib`, `seaborn`, `plotly`
  - R: `tidyverse`, `ggplot2`
- **版本控制**: GitHub Desktop
- **AI 助手**: Google NotebookLM (用於分析文檔與優化代碼)

---

## 📈 學習目標

1. **基礎圖表**: 掌握散佈圖、折線圖、長條圖與箱形圖的繪製。
2. **數據清洗**: 學習在繪圖前使用 Pandas/dplyr 進行資料預處理。
3. **敘事設計**: 提升圖表的可讀性與配色美感。
4. **互動視覺化**: 建立可縮放、可點擊的動態圖表。
